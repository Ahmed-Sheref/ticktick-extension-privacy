Privacy Policy

TickTick Extension

This Privacy Policy explains what information TickTick Extension may collect, how that information is used, and how users can control optional features such as account connection, article saving, weekly email digest, and Telegram quiz delivery.

Effective date: April 18, 2026 Chrome Extension Privacy Policy

1. What the Extension Does

TickTick Extension helps users save articles from the current page, organize them with tags, and generate AI-enhanced notes such as summaries. It also supports optional account connection and optional learning-related features such as weekly email digest and Telegram quiz delivery.

2. Information We May Collect

Page and article data

  • Page title
  • Page URL
  • Selected text
  • Readable article or page text
  • User-entered title, tags, list name, and article content

Account and settings data

  • User ID associated with account connection
  • Email address for optional weekly digest features
  • Connection status and extension preferences
  • Telegram-related settings when the user enables optional Telegram features

Locally stored data

The extension may store temporary page context and user preferences locally in the browser to support saving, reconnecting, and improving the user experience.

3. How We Use Information

  • To save articles and notes requested by the user
  • To organize saved content using tags and lists
  • To generate AI-enhanced summaries and related learning features
  • To support TickTick account connection and related product functionality
  • To send optional weekly email digest content if the user enables it
  • To support optional Telegram quiz delivery if the user enables it
  • To remember settings and improve extension usability

4. Data Sharing and Third-Party Services

We do not sell user data.

Information may be processed by services that are necessary to provide the extension’s functionality, such as backend APIs, account connection services, or optional integrations explicitly enabled by the user.

We do not use or transfer user data for purposes unrelated to the extension’s single purpose.

5. Data Retention

We retain data only for as long as needed to provide the extension’s features, maintain saved content, and support optional features enabled by the user. Some locally stored data may remain in the browser until the user removes it, changes settings, or uninstalls the extension.

6. User Choices and Controls

  • Users can choose what article content to save
  • Users can edit saved data and settings inside the extension
  • Users can enable or disable optional digest and Telegram features
  • Users can uninstall the extension at any time

7. Security

We take reasonable steps to protect user information and to limit access to the data required for the extension’s stated purpose.

TickTick Extension is designed to use user data only to provide the features described in this policy and in the Chrome Web Store listing.

8. Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected on this page with an updated effective date.
